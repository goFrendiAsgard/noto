# Noto

Noto is a personal management tool based on [Zrb](https://pypi.org/project/zrb) and [todo.txt](https://github.com/todotxt/todo.txt).

# How to use

- Fork this repository
- Clone this repository to your local computer
- Start working

    ```bash
    zrb todo list
    ```
